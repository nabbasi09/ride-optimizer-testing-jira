# Ride Optimizer Testing using Jira

## Project Overview

This project focuses on manual testing of a Ride Optimizer application that calculates:

* Shortest path using Dijkstra Algorithm
* Fastest path using A* Algorithm

Jira is used for bug tracking and workflow management.

---

## Objectives

* Validate input fields and system behavior
* Verify shortest and fastest path calculations
* Perform positive and negative testing
* Identify and track bugs using Jira
* Improve software reliability

---

## Technologies Used

* Python (Tkinter for GUI)
* Algorithms: Dijkstra, A*
* Testing: Manual Testing
* Tool: Jira
